# SELECT * on Partitioned Hive Table (COST-PRESTO-001)

**Dimension**: Cost
**Severity**: High
**Scope**: Universal (All Dialects)

## Description
Hive tables can have thousands of partitions and hundreds of columns. SELECT * reads everything, multiplying I/O and cost.
